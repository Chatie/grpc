# GRPC

gRPC Proto Buffers for Chatie

## INSTALL

```shell
$ npm install
...
```

Install Protocol Buffer

### Mac

```shell
$ brew install protoc
...
```

### Linux

```shell
$ sudo apt install protobuf-compiler
...
```
